# ModuleNotFoundError in MAC vsCode

#### 맥 OS 내에서 pip 설치 후 모듈 찾을수 없는 경우 (ModuleNotFoundError)

Mac 운영체제의 경우 기본적으로 pip3이 설치되어 있음

터미널에서 버전까지 입력하여 업데이트를 진행

```termianl
pip3 install --upgrade pip
```
